# Описание:
Этот скрипт позволяет транскрибировать, переворачивать и генерировать комплементарные последовательности для нуклеиновых кислот (ДНК или РНК).

# Запуск и работа команды:
Для запуска подойдёт любой питоновский интерпретатор, программа работает в виде бесконечного цикла. На вход консоль требуется сначала написать команду (см. список команд ниже), а потом последовательность нуклеиновой кислоты.

# Список команд:
exit — завершение исполнения программы
transcribe — напечатать транскрибированную последовательность
reverse — напечатать перевёрнутую последовательность
complement — напечатать комплементарную последовательность
reverse complement — напечатать обратную комплементарную последовательность

# Баги и фичи:
Транскрипция РНК-зависимой РНК-полимеразой не поддерживается (если на вход для транскрипции подается РНК команда работаеть не будет)
Программа сохраняет регистр символов (напр. complement AtGc это TaCg)
Программа работает только с последовательностями нуклеиновых кислот. К примеру, последовательность AUTGC не может существовать. 
Использование в последовательности нуклеиновой кислоты любых символов кроме 'A','G','T','C','U' не поддерживается (alphabetical error)
